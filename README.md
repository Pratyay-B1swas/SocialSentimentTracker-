# Bangladesh News Sentiment Analysis using Python-
Analyzing people's opinions on a specific social event.

# Goal:
"To analyze the overall sentiment of news published in the media about the economy of Bangladesh and determine its trends over time."

# Methods & Techniques:
News API, Python, Pandas, TextBlob, Matplotlib/Seaborn

# Data Analysis Process:
1. Data collection: Retrieving data from News API.
2. Data cleaning: Fixing the description column.
3. Sentiment: Creating Positive/Negative labels using TextBlob.


# Key insights and results:
1. Pie Chart: 77% of articles here were positive or neutral.
2. Bar Chart: The first week saw a surge in positive news, which could be due to a specific event.
